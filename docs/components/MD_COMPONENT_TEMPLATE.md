## DependsOn

このコンポーネントが依存する他のコンポーネントを列挙してください。
例: なし / SearchForm.js, MapComponent.js

- 親コンポーネントから受け取るPropsやデータを記載

- onResults (function)
- searchParams (object)

## Output

- 検索条件と結果を親へ渡す

- 入力が完了したら onSearchParams を呼び出すこと
